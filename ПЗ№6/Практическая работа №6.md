# Практическая работа №6 – Алгоритмы

## Задача 1
### Линейный алгоритм «Перевод времени»

### Алгоритм
1) Ввести `количество часов (hours)`  
2) Если `hours <= 0`, то вывести сообщение  
   `"Количество часов должно быть больше 0"` и завершить алгоритм  
3) Вычислить количество минут:  
   `minutes = hours * 60`  
4) Вывести `minutes`

### Пример работы
1) Ввод: 2 → вывод: `120`  
2) Ввод: 4 → вывод: `240`  
3) Ввод: 0 → вывод: `"Количество часов должно быть больше 0"`  
4) Ввод: -5 → вывод: `"Количество часов должно быть больше 0"`


## Задача 2
### Алгоритм ветвления «Сравнение чисел»

### Алгоритм
1) Ввести числа `A` и `B`  
2) Если `A = B`, то вывести  
   `"Числа равны"` и завершить алгоритм  
3) Если `A > B`, то вывести  
   `"Число A больше"`  
   иначе вывести  
   `"Число B больше"`

### Пример работы
1) Ввод: 2, 4 → вывод: `"Число B больше"`  
2) Ввод: 5, 3 → вывод: `"Число A больше"`  
3) Ввод: 1, 1 → вывод: `"Числа равны"`


## Задача 3
### Алгоритм ветвления «Допуск к экзамену»

### Алгоритм
1) Ввести `текущую оценку за семестр (avgScore)`  
2) Если `avgScore < 1 или avgScore > 5`, то вывести  
   `"Аномальная текущая оценка за семестр!"` и завершить алгоритм  
3) Если `avgScore >= 3`, то вывести  
   `"Допущен"`  
   иначе вывести  
   `"Не допущен"`

### Пример работы
1) Ввод: 5 → вывод: `"Допущен"`  
2) Ввод: 3 → вывод: `"Допущен"`  
3) Ввод: 2 → вывод: `"Не допущен"`  
4) Ввод: 6 → вывод: `"Аномальная текущая оценка за семестр!"`


## Задача 4
### Алгоритм с ветвлением «Скидка в магазине»

### Алгоритм
1) Ввести:
   - `цену за единицу товара (price)`
   - `количество (count)`  

2) Если `price <= 0 или count <= 0`, то вывести  
   `"Некорректные данные для покупки!"` и завершить алгоритм  

3) Вычислить общую сумму:  
   `total = price * count`  

4) Если `total > 1000`, то:  
   `finalTotal = total - total * 0.10`  

   иначе:  
   `finalTotal = total`  

5) Вывести `finalTotal`

### Пример работы
1) Ввод: 200, 6 → вывод: `1080`  
2) Ввод: 100, 5 → вывод: `500`  
3) Ввод: 250, 4 → вывод: `1000`  
4) Ввод: -100, 5 → вывод: `"Некорректные данные для покупки!"`


## Задача 5
### Алгоритм с вложенными ветвлениями «Квадратное уравнение»

### Алгоритм
1) Ввести `a`, `b`, `c`  

2) Если `a = 0`, то:
   - Если `b = 0`, то:
     - Если `c = 0`, вывести  
       `"Бесконечно много решений"`  
     - Иначе вывести  
       `"Нет решений"`  
   - Иначе:
     - `x = -c / b`  
     - Вывести `"Линейное уравнение, x = ..."`  
   - Завершить алгоритм  

3) Вычислить дискриминант:  
   `D = b² - 4ac`  

4) Если `D > 0`, то:
   - `x1 = (-b + √D) / (2a)`  
   - `x2 = (-b - √D) / (2a)`  
   - Вывести `"Два корня: x1, x2"`  

5) Иначе если `D = 0`, то:
   - `x = -b / (2a)`  
   - Вывести `"Один корень: x"`  

6) Иначе:
   - Вывести `"Нет действительных корней"`

### Пример работы
1) Ввод: 1, -3, 2 → вывод: `Два корня: 2 и 1`  
2) Ввод: 1, -2, 1 → вывод: `Один корень: 1`  
3) Ввод: 1, 2, 5 → вывод: `Нет действительных корней`  
4) Ввод: 0, 2, -4 → вывод: `Линейное уравнение, x = 2`  
5) Ввод: 0, 0, 0 → вывод: `Бесконечно много решений`
