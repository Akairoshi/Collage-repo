# Практические задания ПЗ №7

## Задача 1
### Линейный алгоритм «Перевод времени»

### Техническое задание

**Наименование программы:**  
Программа перевода времени из часов в минуты.

**Назначение программы:**  
Перевод заданного количества часов в минуты.

**Функциональное назначение:**  
Программа принимает количество часов, вычисляет количество минут и выводит результат.

**Входные данные:**  
- количество часов (`hours`)

**Выходные данные:**  
- количество минут  
- либо сообщение об ошибке

**Условия работы программы:**  
- если `hours > 0`, то  
  `minutes = hours * 60`  
- иначе вывести  
  `"Количество часов должно быть больше 0"`

**Требования к программе:**  
1. Ввод значения пользователем  
2. Проверка корректности  
3. Вычисление результата  
4. Вывод результата


## Задача 2
### Алгоритм ветвления «Сравнение чисел»

### Техническое задание

**Наименование программы:**  
Программа сравнения двух чисел.

**Назначение программы:**  
Определение, какое из двух чисел больше.

**Функциональное назначение:**  
Программа сравнивает два числа и выводит результат.

**Входные данные:**  
- число `A`  
- число `B`

**Выходные данные:**  
- `"Число A больше"`  
- `"Число B больше"`  
- `"Числа равны"`

**Условия работы программы:**  
- если `A > B` → `"Число A больше"`  
- если `B > A` → `"Число B больше"`  
- если `A = B` → `"Числа равны"`

**Требования к программе:**  
1. Ввод двух чисел  
2. Сравнение значений  
3. Вывод результата


## Задача 3

**Наименование программы:**  
Программа проверки допуска к экзамену.

## Задача 4
### Алгоритм с ветвлением «Скидка в магазине»

### Техническое задание

**Наименование программы:**  
Программа расчёта стоимости покупки.

**Назначение программы:**  
Вычисление итоговой суммы покупки с учётом скидки.

**Функциональное назначение:**  
Программа рассчитывает стоимость покупки и применяет скидку при необходимости.

**Входные данные:**  
- цена (`price`)  
- количество (`count`)

**Выходные данные:**  
- итоговая сумма  
- либо сообщение об ошибке

**Условия работы программы:**  
1. `total = price * count`  
2. если `total > 1000` →  
   `finalTotal = total - total * 0.10`  
3. иначе →  
   `finalTotal = total`  
4. если `price <= 0 или count <= 0` →  
   `"Некорректные данные для покупки!"`

**Требования к программе:**  
1. Ввод данных  
2. Проверка корректности  
3. Расчёт стоимости  
4. Применение скидки  
5. Вывод результата


## Задача 5
### Алгоритм с вложенными ветвлениями «Квадратное уравнение»

### Техническое задание

**Наименование программы:**  
Программа решения квадратного уравнения.

**Назначение программы:**  
Решение уравнения вида `ax² + bx + c = 0`.

**Функциональное назначение:**  
Программа определяет тип уравнения и находит его решения.

**Входные данные:**  
- `a`, `b`, `c`

**Выходные данные:**  
- два корня  
- один корень  
- сообщение об отсутствии решений  
- сообщение о бесконечном количестве решений

**Условия работы программы:**

1. Если `a ≠ 0`:  
   `D = b² - 4ac`  

   - если `D > 0` → два корня  
   - если `D = 0` → один корень  
   - если `D < 0` → нет действительных корней  

2. Если `a = 0`:  
   - если `b ≠ 0` → линейное уравнение  
     `x = -c / b`  
   - если `b = 0 и c = 0` → бесконечно много решений  
   - если `b = 0 и c ≠ 0` → нет решений  

**Требования к программе:**  
1. Обработка всех случаев  
2. Вычисление корней  
3. Корректный вывод результата
